# Система контролю часу та планування задач

Зручний десктопний додаток для керування справами. Дозволяє створювати список задач, встановлювати дедлайни та відстежувати час виконання роботи.

## Функціонал
* **Задачі:** Створення завдань із категоріями (Навчання, Робота, Особисте) та датами виконання.
* **Таймер:** Вбудований секундомір для трекінгу часу по кожній задачі.
* **Сортування:**
    * **Термінові:** показує спочатку прострочені та найближчі задачі.
    * **За назвою:** сортування за алфавітом.
* **Статистика:** Панель із загальним часом роботи та кількістю виконаних справ.
* **Збереження:** Всі дані автоматично зберігаються (JSON), нічого не зникає після закриття.

## Технології
Проект розроблено мовою **C# (WPF)** із використанням принципів об'єктно-орієнтованого програмування (**ООП**) та архітектури **MVVM**.

## Як запустити
1.  Відкрити проект у Visual Studio.
2.  Натиснути **Start** або клавішу **F5**.
